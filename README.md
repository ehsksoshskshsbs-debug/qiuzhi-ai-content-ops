# 秋芝 AI 内容运营作品集

本项目是一套面向“秋芝2046”内容运营岗位的求职作品集，包含可调用的 Codex Skill、运营知识库、完整模拟案例和公开展示网站。

## 招聘投递成品

- 独立作品页：`/portfolio`
- 7页 PDF：`site/public/qiuzhi-ai-content-ops-portfolio.pdf`
- 结构化内容底座：`site/data/portfolio.json`
- PDF 生成脚本：`scripts/generate_portfolio_pdf.py`

作品页包含一页账号观察、3个选题、五平台完整样稿、7天发布排期、14条模拟反馈和前瞻复盘框架。

项目不代表秋芝2046官方。账号信息来自公开页面；评论、私信和后台指标均为模拟数据。

主要目录：

- `agent-skill/qiuzhi-content-ops`：Codex Agent 源文件；
- `knowledge-base`：人类可读的运营知识库；
- `case-study`：模拟案例和台账数据；
- `site`：公开作品集网站；
- `docs`：设计与实施规格。
