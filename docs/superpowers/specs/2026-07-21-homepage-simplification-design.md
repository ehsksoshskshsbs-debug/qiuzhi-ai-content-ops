# 首页精简设计

## 目标

将公开首页聚焦为“快速了解秋芝2046账号与代表视频”的单一用途，删除与求职作品集、Agent、知识库、模拟案例、运营看板和云端台账有关的首页入口与展示内容。

## 保留内容

- 顶部品牌与导航；导航只保留“账号洞察”和“视频速览”。
- 首页开场区域，但文案与数据只描述账号研究和视频分类。
- 账号洞察区。
- 30 条代表视频速览、搜索、分类、难度与观看路线。
- 可信边界说明与页脚，并删除其中的 Agent 求职作品集表述。
- `/ops` 云端反馈工作台及其数据库、接口、登录保护和直达地址。

## 删除内容

- 顶部导航中的 Agent、知识库、模拟案例、运营看板和云端台账。
- 首页 Agent 能力区、知识库区、模拟案例区和运营看板区。
- 开场区域中的 Agent 按钮、Agent 能力统计和相关介绍。
- 页脚及页面元数据中与 Agent、知识库、案例和运营看板有关的文案。
- 删除后不再使用的首页状态、常量和交互逻辑。

## 交互与路由

- “账号洞察”和“视频速览”继续使用页内锚点跳转。
- 视频搜索、分类、难度和观看路线维持原有行为。
- `/ops` 不从公开首页暴露入口，但知道地址且满足登录白名单的用户仍可访问。

## 验收标准

- 桌面端与移动端顶部导航仅出现“账号洞察、视频速览”。
- 公开首页不再出现 Agent、知识库、模拟案例、运营看板和云端台账入口或内容区。
- 视频筛选功能正常，公开首页可正常构建和访问。
- `/ops` 路由、鉴权、D1 数据与 API 不受影响。
