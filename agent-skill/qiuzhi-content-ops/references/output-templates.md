# 输出模板

## 选题卡

```markdown
# [内容编号] 选题名称
- 一句话结果：
- 核心用户：
- 高频痛点：
- 为什么现在做：
- 可视化演示：
- 差异化角度：
- 六维评分：痛点 / 结果 / 匹配 / 演示 / 差异 / 时效风险
- 风险与待核验：
- 建议结论：进入制作 / 优化后制作 / 观察
```

## 多平台内容包

```markdown
# 主叙事
- 内容编号：
- 核心承诺：
- 事实底座：
- 目标用户：

## B站
标题 / 封面字 / 章节 / 口播 / 演示 / 简介 / 置顶评论

## 抖音
三秒钩子 / 口播 / 镜头 / 字幕 / 互动

## 视频号
场景开场 / 口播 / 步骤 / 边界 / 转发引导

## 小红书
标题 / 封面 / 八页卡片 / 正文 / 关键词 / 评论问题

## 公众号
标题 / 摘要 / 导语 / 完整步骤 / FAQ / 结语
```
