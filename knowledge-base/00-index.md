# 运营知识库索引

本知识库服务于 AI 知识博主的日常内容运营与求职作品集展示。

## 使用顺序

1. 用 `01-account-and-platforms.md` 判断账号定位和平台表达；
2. 用 `02-content-sop.md` 完成选题、生产、发布和复盘；
3. 用 `03-feedback-and-campaigns.md` 整理用户声音并管理活动节点；
4. 用 `04-templates.md` 建立可追溯台账。

## 数据边界

- 秋芝2046账号信息来自公开页面，查询日期为 2026-07-21；
- 账号定位之外的用户画像属于内容分析推断；
- 案例评论、私信和运营指标全部为模拟数据；
- 本项目不代表秋芝2046官方。
